    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1><a id="user-content-brokenview" class="anchor" href="#brokenview" aria-hidden="true"><span class="octicon octicon-link"></span></a>BrokenView</h1>

<p>Glass-break effect for views.<br></p>

<p><a href="https://raw.githubusercontent.com/zhanyongsheng/raw/master/BrokenView/image/demo.gif" target="_blank"><img src="https://raw.githubusercontent.com/zhanyongsheng/raw/master/BrokenView/image/demo.gif" alt="brokenview" style="max-width:100%;"></a></p>

<h1><a id="user-content-demo" class="anchor" href="#demo" aria-hidden="true"><span class="octicon octicon-link"></span></a>Demo</h1>

<p><a href="https://raw.githubusercontent.com/zhanyongsheng/raw/master/BrokenView/apk/demo.apk">Download APK</a></p>

<h1><a id="user-content-usage" class="anchor" href="#usage" aria-hidden="true"><span class="octicon octicon-link"></span></a>Usage</h1>

<h3><a id="user-content-android-studio" class="anchor" href="#android-studio" aria-hidden="true"><span class="octicon octicon-link"></span></a>Android Studio</h3>

<div class="highlight highlight-source-groovy-gradle"><pre><span class="pl-en">dependencies</span> {
   compile <span class="pl-s"><span class="pl-pds">'</span>com.zys:brokenview:1.0.3<span class="pl-pds">'</span></span>
}</pre></div>

<h3><a id="user-content-eclipse" class="anchor" href="#eclipse" aria-hidden="true"><span class="octicon octicon-link"></span></a>Eclipse</h3>

<p>Just put <a href="https://raw.githubusercontent.com/zhanyongsheng/raw/master/BrokenView/jar/brokenview.jar">brokenview.jar</a> into the libs folder of your app.</p>

<h3><a id="user-content-basic-usage" class="anchor" href="#basic-usage" aria-hidden="true"><span class="octicon octicon-link"></span></a>Basic Usage</h3>

<div class="highlight highlight-source-java"><pre><span class="pl-smi">BrokenView</span> brokenView <span class="pl-k">=</span> <span class="pl-smi">BrokenView</span><span class="pl-k">.</span>add2Window(context);
<span class="pl-smi">BrokenTouchListener</span> listener <span class="pl-k">=</span> <span class="pl-k">new</span> <span class="pl-smi">BrokenTouchListener</span>.<span class="pl-smi">Builder</span>(brokenView)<span class="pl-k">.</span>build();
view<span class="pl-k">.</span>setOnTouchListener(listener);</pre></div>

<h1><a id="user-content-more-config" class="anchor" href="#more-config" aria-hidden="true"><span class="octicon octicon-link"></span></a>More Config</h1>

<p>Use <code>BrokenTouchListener.Builder</code> for more configurations, and set custom attributes as shown below:</p>

<div class="highlight highlight-source-java"><pre><span class="pl-smi">BrokenTouchListener</span><span class="pl-k">.</span>Builder(brokenView)<span class="pl-c1">.</span>
    setComplexity(<span class="pl-c1">...</span>)<span class="pl-c1">.</span>          <span class="pl-c">// default 12 </span>
    setBreakDuration(<span class="pl-c1">...</span>)<span class="pl-c1">.</span>       <span class="pl-c">// in milliseconds, default 700ms</span>
    setFallDuration(<span class="pl-c1">...</span>)<span class="pl-c1">.</span>        <span class="pl-c">// in milliseconds, default 2000ms</span>
    setCircleRiftsRadius(<span class="pl-c1">...</span>)<span class="pl-c1">.</span>   <span class="pl-c">// in dp, default 66dp, you can disable circle-rifts effect by set it to 0</span>
    setEnableArea(<span class="pl-c1">...</span>)<span class="pl-c1">.</span>          <span class="pl-c">// set the region or childview that can enable break effect,</span>
                                 <span class="pl-c">// be sure the childView or childView in region doesn't intercept any touch event</span>
    setPaint(<span class="pl-c1">...</span>)<span class="pl-c1">.</span>               <span class="pl-c">// the paint to draw rifts</span>
    build();</pre></div>

<p>You can also set a callback to listen to the status of animations, like this:</p>

<div class="highlight highlight-source-java"><pre>brokenView<span class="pl-k">.</span>setCallback<span class="pl-k">.</span>(<span class="pl-k">new</span> <span class="pl-smi">BrokenCallback</span>() {
   <span class="pl-k">@Override</span>
   <span class="pl-k">public</span> <span class="pl-k">void</span> <span class="pl-en">onStart</span>(<span class="pl-smi">View</span> <span class="pl-v">v</span>) {<span class="pl-c1">...</span>}
   <span class="pl-k">@Override</span>
   <span class="pl-k">public</span> <span class="pl-k">void</span> <span class="pl-en">onCancel</span>(<span class="pl-smi">View</span> <span class="pl-v">v</span>) {<span class="pl-c1">...</span>}
   <span class="pl-k">@Override</span>
   <span class="pl-k">public</span> <span class="pl-k">void</span> <span class="pl-en">onCancelEnd</span>(<span class="pl-smi">View</span> <span class="pl-v">v</span>) {<span class="pl-c1">...</span>}
   <span class="pl-k">@Override</span>
   <span class="pl-k">public</span> <span class="pl-k">void</span> <span class="pl-en">onRestart</span>(<span class="pl-smi">View</span> <span class="pl-v">v</span>) {<span class="pl-c1">...</span>}
   <span class="pl-k">@Override</span>
   <span class="pl-k">public</span> <span class="pl-k">void</span> <span class="pl-en">onFalling</span>(<span class="pl-smi">View</span> <span class="pl-v">v</span>) {<span class="pl-c1">...</span>}
   <span class="pl-k">@Override</span>
   <span class="pl-k">public</span> <span class="pl-k">void</span> <span class="pl-en">onFallingEnd</span>(<span class="pl-smi">View</span> <span class="pl-v">v</span>) {<span class="pl-c1">...</span>}
});</pre></div>
